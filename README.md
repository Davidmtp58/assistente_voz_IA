# 🎙️ Assistente de Voz com IA

Projeto desenvolvido durante o Bootcamp GenAI da DIO + Bradesco.

## 📌 Sobre
Sistema de conversação por voz que usa IA para entender perguntas
e responder com voz em português.

## 🧰 Tecnologias
- Whisper (OpenAI) — voz para texto
- ChatGPT (OpenAI) — geração de respostas
- gTTS (Google) — texto para voz
- Google Colab — ambiente de execução

## ▶️ Como Usar
1. Abra o notebook no Google Colab
2. Configure sua chave da OpenAI nos Secrets do Colab
3. Execute as células 1 e 2 uma vez
4. Execute a Célula 3 sempre que quiser perguntar
